# Project 3 – Interactive Portfolio Landing Page

🔗 Live Demo: https://decode-labs-project03.vercel.app  


##  Overview
This is **Project 3 (Frontend Interactive Web Project)** built as part of my web development learning journey.  
The goal of this project is to demonstrate **DOM manipulation, responsive UI design, and JavaScript interactivity**.

It is a modern **personal portfolio landing page** designed with a clean UI and smooth user experience.


##  Features

-  Dark Mode Toggle (with localStorage save)
-  Fully Responsive Design (Mobile + Tablet + Desktop)
-  Mobile Navigation Menu
-  Smooth Section Navigation
-  Modern Color Palette UI
-  Clean and Optimized JavaScript
-  DOM Manipulation Practice
-  Portfolio Sections (About, Skills, Projects, Contact)
-  Contact Form UI
-  Reusable Components Structure

---

##  Tech Stack

- HTML5  
- CSS3 (Custom Design System)  
- JavaScript (Vanilla JS)  
- Font Awesome Icons  
- Google Fonts (Poppins)  
- Vercel (Deployment)
